# Github-Search

This is a web application that allows the user to search for existing GitHub accounts and view their public repositories as well.

[![Screenshot-2021-11-29-at-10-37-24.png](https://i.postimg.cc/bJgph3D6/Screenshot-2021-11-29-at-10-37-24.png)](https://postimg.cc/gXxfyyKR)

## Technologies Used

- Angular and Github-Api.


##### Setup Instructions and Installation

- Fork this repository in to your own.
- Run the command `git clone https://github.com/TimKirathe/github-search.git` in your terminal to clone the repository into your local one.
- Navigate to the `github-search` file created and open it in your terminal.
- Once opened, run the command `ng serve --open` in the terminal to open the application on a live server.
